## 1. НАЗНАЧЕНИЕ И ЦЕЛИ ПРОЕКТА

* **Назначение:** Программный комплекс для генерации реалистичного трехмерного рельефа местности (ландшафта) с использованием математического аппарата рядов и преобразований Фурье.
* **Цель:** Продемонстрировать практическое применение спектрального анализа для моделирования природных фрактальных структур (розового и фрактального шума), реализовать бесшовную генерацию мира и интерактивный 3D-рендер.

## 2. МАТЕМАТИЧЕСКОЕ ЯДРО (СПЕКТРАЛЬНЫЙ СИНТЕЗ)

Генерация геометрии должна базироваться на концепции спектрального разложения. Алгоритм обязан поддерживать два режима работы:

### 2.1. Режим А: Быстрое Преобразование Фурье (2D FFT)

Применяется для генерации изолированных локальных карт (карт высот).

1. **Генерация шума:** Создание исходной матрицы белого шума $W$ размером $N \times N$ (где $N = 2^k$ для оптимизации FFT).
2. **Прямое преобразование:** Перевод матрицы в частотную область:

$$S = \text{fftshift}(\text{fft2}(W))$$


3. **Частотная фильтрация (1/f фильтр):** Каждая спектральная компонента с координатами $(u, v)$ умножается на фильтр спадания амплитуды:

$$H(u, v) = S(u, v) \cdot \frac{1}{(u^2 + v^2)^{\beta/2}}$$



*Где $\beta$ — спектральный индекс (параметр шероховатости ландшафта).*
4. **Обратное преобразование:** Возврат в пространственный домен для получения матрицы высот $Z$:

$$Z = \text{real}(\text{ifft2}(\text{ifftshift}(H)))$$


5. **Нормализация:** Приведение значений матрицы $Z$ к диапазону $[0, 1]$.

### 2.2. Режим Б: Аналитический метод гармоник (Бесконечные бесшовные чанки)

Применяется для генерации бесконечного мира, разбитого на сетку чанков размера $M \times M$.

1. **Инициализация спектра:** На основе фиксированного базового зерна (*Seed*) генерируется массив из $K$ случайных волновых векторов $\vec{k}_i = (k_{xi}, k_{yi})$ и случайных фазовых сдвигов $\phi_i$.
2. **Расчет по координатам:** Высота любой точки ландшафта с глобальными координатами $(x, y)$ рассчитывается аналитически как сумма гармоник:

$$Z(x, y) = \sum_{i=1}^{K} A_i \cdot \sin(k_{xi}x + k_{yi}y + \phi_i)$$



Амплитуда $A_i$ жестко привязана к модулю частоты $|\vec{k}_i|$ по закону обратной пропорциональности ($1/f^\beta$).
3. **Бесшовность:** За счет того, что функция $Z(x, y)$ непрерывна, расчет соседних чанков (например, сдвиг по сетке на величину $M$) гарантирует идеальное совпадение геометрии на стыках без швов.

## 3. ТРЕБОВАНИЯ К 3D-ВИЗУАЛИЗАЦИИ И ПОСТ-ОБРАБОТКЕ

Программа не просто строит абстрактную сетку, а превращает карту высот в полноценную трехмерную сцену.

### 3.1. Генерация 3D-Меша (Полигональной сетки)

* Координаты $X$ и $Y$ формируются в виде регулярной двумерной сетки (`np.meshgrid`).
* Значения матрицы $Z$ аппроксимируются как высота вершин по оси ординат.

### 3.2. Процедурное текстурирование (Карта биомов)

Цвет каждой вершины меша вычисляется динамически в зависимости от значения высоты $Z \in [0, 1]$:

* $Z < 0.25$ — Глубоководный океан (Темно-синий)
* $0.25 \le Z < 0.32$ — Мелководье / Пляж (Бирюзовый / Желтый песок)
* $0.32 \le Z < 0.65$ — Долины / Равнины (Насыщенный зеленый)
* $0.65 \le Z < 0.85$ — Горные склоны / Скалы (Серый)
* $Z \ge 0.85$ — Ледники и вечные снега (Чисто белый)

### 3.3. Расчет честного 3D-освещения (Тени на склонах)

Для придания ландшафту объема необходимо реализовать расчет затененности по модели Ламберта:

1. Вычисляются частные производные матрицы высот для нахождения вектора нормали $\vec{n}$ в каждой вершине:

$$\vec{n} = \left(-\frac{\partial Z}{\partial x}, -\frac{\partial Z}{\partial y}, 1\right)$$


2. Задается вектор направления «солнечного света» $\vec{L} = (x_l, y_l, z_l)$.
3. Интенсивность освещения вершины определяется как скалярное произведение: $I = \max(0, \vec{n} \cdot \vec{L})$. Цвет биома в данной точке умножается на коэффициент $I$.

## 4. ТРЕБОВАНИЯ К ИНТЕРФЕЙСУ И ИНТЕРАКТИВНОСТИ

Среда разработки — **Jupyter Notebook**. Проект должен содержать интерактивную панель управления на базе `ipywidgets` и графического движка **Plotly 3D**.

Пользователь должен иметь возможность крутить слайдеры в реальном времени:

* **Seed (Зерно генератора):** Мгновенная перегенерация случайного ландшафта.
* **Спектральный индекс ($\beta$):** Изменение характера местности (от пологих холмов при $\beta=2.5$ до острых скал при $\beta=1.1$).
* **Количество октав/гармоник:** Демонстрация детализации (при малом числе — гладкие волны, при большом — прорисовываются мелкие камни).
* **Положение солнца (Угол $\theta$):** Динамическое скольжение теней по 3D-горам.

Рендер Plotly должен позволять пользователю вращать ландшафт мышкой, приближать (zoom) и рассматривать структуру долин в полноценном 3D-пространстве.
