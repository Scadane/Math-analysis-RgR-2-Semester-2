# 🏔️ Процедурная генерация 3D-ландшафта

Интерактивный генератор рельефа на основе спектрального синтеза и аналитического метода гармоник. Расчётно-графическая работа по математическому анализу.

## Возможности

- **Спектральный метод** — генерация карты высот через 2D FFT с фильтрацией по закону $1/f^\beta$
- **Аналитический метод** — бесшовная генерация мира через суммирование гармоник
- **Мульти-чанк** — генерация области (1×1, 3×3 или 5×5) чанков с плавным сшиванием
- **Освещение** — интерактивное управление солнцем (высота, азимут), тени по Ламберту
- **Биомы** — процедурное текстурирование (вода, песок, равнины, скалы, снег)
- **Визуализация спектров** — наглядное отображение частотной фильтрации

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

Откроется в браузере на `http://localhost:8501`.

## Структура

```
app.py                   — веб-приложение Streamlit
landscape_synthesis.py   — математическое ядро (FFT, гармоники, нормали, освещение, биомы)
main.py                  — консольная проверка
test_math.py             — тесты математических функций
landscape_notebook.ipynb — Jupyter-ноутбук с теорией и интерактивными демонстрациями
```
